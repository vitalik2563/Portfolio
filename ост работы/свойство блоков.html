Блочная модель CSS

На веб-странице любому HTML элементу соответствует определенная прямоуголная область(блок). Во время рендеринга браузер определяет размеры и положение этих блоков, а также определяет их CSS свойства, чтобы применить их и отобразить на экране.

В CSS создана специальная блочная модель (фнгл.box model), которая описывает, из чего состоит любой блок и от каких свойств зависят его размеры. В этой модели каждый блок состоит из четырех областейЖ margin (внешние отступы), border (граница элемента), padding (внутринние отступы), content (содержимое).
